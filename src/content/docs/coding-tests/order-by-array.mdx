---
title: Order by multiple keys
description: Sort an array of objects by multiple keys
---

Origin array:

```javascript
const arr = [
  {name: 'John', a: 1, b: 2, c: 3},
  {name: 'Jane', a: 1, b: 3, c: 4},
  {name: 'Janet', a: 2, b: 1, c: 2},
  {name: 'Janet', a: 2, b: 3, c: 1},
  {name: 'Janet', a: 2, b: 3, c: 2},
];
```

Sort by

```typescript
[{
    key: 'a',
    order: 'asc'
}, {
    key: 'b',
    order: 'desc'
}, {
    key: 'c',
    order: 'asc'
}]
```

Implement a function to sort the array by the given keys.

```javascript
function orderBy(arr, keys) {
  return arr.sort((a, b) => {
    for (let i = 0; i < keys.length; i++) {
      const key = keys[i].key;
      const order = keys[i].order;
      if (a[key] < b[key]) {
        return order === 'asc' ? -1 : 1;
      } else if (a[key] > b[key]) {
        return order === 'asc' ? 1 : -1;
      }
    }
    return 0;
  });
}
```
